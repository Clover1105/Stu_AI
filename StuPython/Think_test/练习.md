### 数字与运算

1. **整数运算**
   编写程序，计算 `(25 + 38) * 4 / 2` 的结果，并打印输出。
2. **取余与整除**
   输入一个三位数（如 `367`），输出它的百位数、十位数和个位数。
   *提示：使用整除 // 和取余 % 运算。*
3. **角度计算**
   已知直角三角形的两条直角边分别为 `3` 和 `4`，求斜边的长度（保留两位小数）。
   *提示：使用 math.sqrt() 或 \**0.5。*
4. **数字进制转换**
   输入一个十进制整数，输出它的二进制、八进制和十六进制表示。
   *提示：使用 bin()、oct()、hex()。*

### 字符串操作

1. **字符串拼接与格式化**
   定义三个变量 `name = "张三"`、`age = 18`、`city = "北京"`，使用 f-string 输出：
   `我是张三，今年18岁，来自北京。`
2. **字符串索引与切片**
   给定字符串 `s = "Python Programming"`，分别取出：
   - 第一个字符
   - 最后一个字符
   - 前5个字符
   - 从索引3开始到结尾的字符
3. **字符串常用方法**
   给定字符串 `" hello world "`，完成以下操作：
   - 去掉两端空格
   - 转换为全大写
   - 将 `world` 替换为 `Python`
   - 按空格分割成列表
4. **查找子串**
   给定字符串 `"abcabcabc"`，找出第二个 `"abc"` 的起始索引。如果不存在则输出 `-1`。
   *提示：使用 find() 方法。*

### 输入与输出

1. **用户输入与类型转换**
   编写程序，接收用户输入的年龄，判断是否大于等于18岁，并输出对应的提示信息。
2. **格式化输出**
   接收用户输入的姓名和身高（米），输出：
   `姓名：xxx，身高：x.xx米`
   身高保留两位小数。
