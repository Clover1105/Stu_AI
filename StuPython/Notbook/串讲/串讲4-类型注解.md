Python 类型注解 (Type Hints / Annotations)

Python 3.12+ 现代化写法

Python 3.5 引入类型注解，经过多版本迭代，3.12 已大幅简化。

类型注解不会影响代码运行，但可以：

1. 提高代码可读性
2. 配合 IDE 提供智能提示
3. 配合工具做静态类型检查

运行环境要求：

- Python 3.12+
- typing 模块（标准库自带）

类型注解 主要针对变量和函数的参数，返回值 做出类型约定

```python
def fn(x):
	print(x[1])
fn(def)	# 有类型注解后，会有红色波浪提醒未来运行会报错
```

# 二、变量类型注解

全部使用内置泛型 + | 联合语法

```python
# 基本类型
name: str = "张三"
age: int = 25
height: float | int = 1.75
is_student: bool = False

# 列表 / 字典 / 元组 / 集合 -- 直接用内置类型，不再从 typing 导入
scores: list[int] = [90, 85, 92]
names: list[str] = ["Alice", "Bob"]
matrix: list[list[int]] = [[1, 2], [3, 4]]

user_info: dict[str, Any] = {"name": "张三", "age": 25}
config: dict[str, str] = {"host": "localhost", "port": "8080"}

coordinate: tuple[int, int] = (100, 200)
record: tuple[str, int, float | int ] =("张三", 25, 1.75)

unique_ids: set[int] = {1, 2, 3, 4}

# X | None 替代 Optional[X]
nickname: str | None = None
nickname = "小明"

# X | Y 替代 Union[X, Y]
code: int | str = 200
code = "OK"

# Any：任意类型（相当于关闭类型检查）
data: Any {"key": "value"}
data: [1, 2, 3]

# Final：标记为常量
MAX_SIZE: Final[int] = 1024

# Literal:只能是给定的字面量之一
mode: Literal["r","w","a"] = "r"
mode: Literal[100, "hello", True] = 100

HttpMethod = Literal["GET","POST", "PUT", "DELETE"]
```

@override 装饰器（3.12 新增，PEP 698）：明确标记方法覆写

```python
class Animal:
    def speak(self) -> str:
    	return
class Dog(Animal):
    @override 1个用法
    def speak(self) -> str:

        return "汪汪"
```

# 六、泛型类

新式语法（3.12新增，PEP695）

旧式: T = TypeVar("T"); class Stack(Generic[T]):

```python
class Stack[T]: 
	"""泛型栈：T 是类型参数"""
	def __init__(self) -> None:
		self._items: list[T] = []
	def push(self, item: T) -> None: 
		self._items.append(item)
	def pop(self) -> T | None: 
		return self._items.pop() if self._items else None
	def peek(self) -> T | None:
		return self._items[-1] if self._items else None
	@property 
	def size(self) -> int:
		return len(self._items)
```

```python
class Pair[K, V]:
	"""泛型键值对：多个类型参数"""
	def __init__(self, key: K, value: V) -> None:
    self.key: K = key
    self.value: V = value
    
    def swap(self) -> "Pair[V, K]":
	"""交换键值，返回新的 Pair"""
	return Pair(self.value, self.key)

#受限泛型：约束类型参数的上界
class NumericContainer[ out T: (int, float int )]:
    """类型参数约束：T 只能是 int 或 float"""
    def __init__(self, value: T) -> None:
    	self.value: T = value
    def double(self) -> T:
    	return self.value * 2 # type: ignore[return-value]
```

泛型函数（3.12 新语法）

```python
def first[T](items: list[T]).-> T | None: 
    """泛型函数：返回列表第一个元素"""
    return items[0] if items else None
def count_occurrences[T](items: list[T]) -> dict[T, int]: 
    """泛型函数：统计每个元素出现次数"""
    result: dict[T, int] = {}
    for item in items:
    	result[item] = result.get(item,0)+1
    return result
```
