### 层次讲解

整个页面分为3个层：

#### 结构层：

快捷方式：在 `HBuilderX `中输入 `u + 标签名`（如 `uimage`、`uview`），按回车键（或 Tab 键）即可快速生成对应的 `uni-app` 标签模板结构

`template` ：是 **Vue 的容器标签**，用于包裹多个元素，但**不会渲染为真实的 DOM 节点**；**不可见的逻辑容器**，用于 v-if/v-for，不渲染

`view` ：是 `uni-app` 的**视图容器组件**，相当于 HTML 的 `<div>`，是最基础的**块级元素**；**可见的布局容器**，相当于 div，用于排版和样式

`swiper` ：是 `uni-app` 中内置的**轮播图容器组件**，配合 `swiper-item` 使用，用于实现图片或内容的滑动切换效果，还支持自动播放、指示点、循环滑动等常见功能



![5-uswiper](大模型项目实战1.assets/image-20260716112748296.png)

`image`：是 **uni-app** 中用于**显示图片**的基础组件，相当于 HTML 中的 `img` 标签，但功能更强大，核心优势是内置了 `mode` 属性，无需额外 CSS 即可实现图片的缩放和裁剪适配，配合懒加载和错误处理，能很好地应对多端开发中的图片展示需求

![5-uimage](大模型项目实战1.assets/image-20260716113128689.png)

准备3个图片，在网上找3张图片，图片路径放在`src=“”`中

![5-准备图片位置](大模型项目实战1.assets/image-20260716113744958.png)

![5-放入图片](大模型项目实战1.assets/image-20260716135714549.png)

#### 表现层：

设置宽度，在`swiper`标签中添加`class`类选择器，`class="myswiper"`

![6-添加类名](大模型项目实战1.assets/image-20260716142353747.png)

在`style`表现层中添加对应类的样式

`style` ：在` uni-app/Vue` 中用于**定义组件的样式**，相当于 HTML 中的 `<style>` 标签，但增加了 **scoped** 等特性。

> 核心属性：
>
> scoped：样式仅作用于当前组件，不污染全局
>
> lang：指定预处理器，如 scss、less、stylus

![6-添加样式](大模型项目实战1.assets/image-20260716143521203.png)
